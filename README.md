# NiroService App

אפליקציית שירות ללקוחות נירומטל לפתיחת קריאות שירות לציוד מטבח תעשייתי ומוסדי בצורה מהירה וברורה.  
האפליקציה מרכזת את פרטי הלקוח והציוד, מאפשרת צירוף מדיה, ומספקת מעקב סטטוס מול מוקד השירות.  

## למה זה קיים

כדי לצמצם חיכוך בפתיחת קריאה.  
כדי לאסוף מידע נכון כבר בהתחלה ולחסוך טלפונים חוזרים.  
כדי לתת ללקוח שקיפות על התהליך והתקדמות הטיפול.  

## יכולות עיקריות

• פתיחת קריאת שירות עם שדות חובה מסודרים.  
• צירוף תמונות ומסמכים רלוונטיים.  
• קבלת מספר קריאה ואישור פתיחה.  
• מסך מעקב סטטוס ועדכונים.  
• הכוונה לטכנאי או למוקד לפי סוג התקלה.  

## טכנולוגיות

Front End מבוסס HTML CSS JavaScript.  
אופציה לשדרוג ל PWA כדי לאפשר התקנה במסך הבית והתראות.  
אופציה לחיבור ל Backend לצורך שמירה וניהול קריאות.  

## התקנה והרצה מקומית

1. מתקינים VS Code.  
2. פותחים את הפרויקט בתיקיה המקומית.  
3. מריצים עם תוסף Live Server או פותחים את index.html בדפדפן.  

## מבנה תיקיות מומלץ

• index.html  
• css

## הפצה מהירה עם GitHub Pages

1. נכנסים לריפו ב GitHub.  
2. עוברים ל Settings.  
3. נכנסים ל Pages.  
4. בוחרים Deploy from a branch.  
5. בוחרים branch בשם main.  
6. בוחרים תיקיה root.  
7. שומרים וממתינים שיופיע קישור לאתר.  

## חיבור למערכת שירות

יש שתי אפשרויות נפוצות.  
אפשרות אחת היא אינטגרציה ישירה עם מערכת הקריאות הקיימת דרך API.  
אפשרות שנייה היא פתרון ביניים ששולח את נתוני הקריאה למייל מוקד או לטופס שמזין גיליון.
